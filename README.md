Dashboard de Receita – Xbox Game Revenue 2024


- Descrição do Projeto

Este projeto foi desenvolvido como parte de um desafio da DIO (Digital Innovation One), utilizando uma base de dados fornecida pela própria plataforma.
O objetivo principal foi criar um dashboard interativo no Excel para análise de faturamento do Xbox ao longo do ano de 2024.


- Objetivo

Analisar o faturamento mensal

Identificar o mês com maior receita

Avaliar impacto de cupons

Criar uma visualização clara e executiva para apoio à tomada de decisão


- Estrutura do Arquivo Excel

O arquivo está organizado em 4 abas principais, seguindo boas práticas de organização e manutenção:

Assets: Recursos visuais do projeto, Paleta de cores, Ícones e elementos gráficos utilizados no dashboard

Base: Base de dados original disponibilizada pela DIO, Contém informações de assinaturas, valores, datas, cupons e add-ons

Cálculos: Tabelas Dinâmicas utilizadas para cálculo dos KPIs, Base para todos os indicadores e gráficos do dashboard

Dashboard: Visual final do projeto, KPIs de faturamento, Gráficos de receita mensal, Segmentações para filtros interativos


- Principais Indicadores (KPIs)

Receita Total

Faturamento Mensal

Faturamento por assinaturas

Impacto dos Cupons

Identificação do mês com maior faturamento


- Ferramentas Utilizadas

Microsoft Excel 365

Tabelas Dinâmicas

Segmentações de Dados

Gráficos Dinâmicos

Design orientado a dashboard executivo


- Resultado

O projeto entrega um dashboard funcional, organizado e visualmente intuitivo, permitindo identificar rapidamente padrões de faturamento, meses de maior desempenho e impacto de descontos e produtos adicionais.
